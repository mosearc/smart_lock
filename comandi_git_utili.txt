git init -> inizializza git nella cartella
git clone https://.... -> copia una cartella in locale e setta gia init
git status -> stato del git
gh auth login -> per loggarmi *basta una volta per pc*
crei la repo su github e copi il link https
git remote add origin <link> -> linka tutto *con clone non serve*
git pull origin <nome-branch> -> mette in locale il branch chiesto
git add <nome-file> -> aggiunge il file
git commit -m "mex" -> committa com messaggio
git push origin <nome-branch> -> pusha tutto su github
git checkout <nome-branch> -> switcha branch
git branch <new-branch> -> crea un nuovo branch da quello corrente se ci aggiungi <base-branch> lo crea da quello base
